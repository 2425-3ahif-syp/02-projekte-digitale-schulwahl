== Beschreibung

Die Digitale Schulwahlplattform ist eine Anwendung zur Modernisierung der Schulsprecherwahlen an der HTL Leonding.
Sie ersetzt den bisherigen papierbasierten Wahlprozess durch ein digitales System.
